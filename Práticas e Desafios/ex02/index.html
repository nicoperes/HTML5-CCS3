Tags em HTML5

Como linguagem de marcação, em HTML5 há muitas opções para um tipo específico de formatação de texto, desde parágrafos, títulos, entre outros. Abaixo podemos observar algumas das mais comuns e utilizadas tags para se usar ao programar em HTML5:

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Código-Fonte
  • Citações
  • Listas Ordenadas/Númeradas
  • Listas Não-Ordenadas/Demarcada
